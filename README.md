# nuetzliches/homebrew-tap

Homebrew tap for [nuetzliches](https://github.com/nuetzliches) CLI tools.

## Usage

```sh
brew install nuetzliches/tap/composit
```

Or tap once, then install by name:

```sh
brew tap nuetzliches/tap
brew install composit
```

## Formulae

| Formula | Description |
|---------|-------------|
| [`composit`](Formula/composit.rb) | Governance-as-Code for AI-generated infrastructure |
| [`croniq`](Formula/croniq.rb) | Distributed job scheduler with cron, calendar, and runner support |

Formulae are updated automatically by each project's release workflow.
